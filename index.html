Course Overview
What you'll learn
Master the fundamentals of Artificial Intelligence, Machine Learning, Generative AI, Responsible AI, prompt engineering concepts, and the AWS services commonly used to build intelligent cloud applications.

Learn the concepts, terminology, AWS best practices, and exam-oriented knowledge required to confidently prepare for the AWS Certified AI Practitioner (AIF-C01) certification.
