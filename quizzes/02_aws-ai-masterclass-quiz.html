MC 1MCA1
A news agency uses AI to detect sentiment in reader comments. A media startup uses AI to write full feature articles from a topic brief. Which statement correctly maps each task to the appropriate AI paradigm?
ABoth tasks are Generative AI — they both process natural language and produce an output from input data.
BSentiment detection is traditional ML (analyzes existing content, produces a classification); article writing is Generative AI (creates new content from a brief).
CBoth tasks are traditional ML — sentiment and article generation both involve pattern recognition from training data.
DSentiment detection is Generative AI because it generates a label; article writing is traditional ML because it follows learned templates.
MC 2MCB1–B2
A manufacturing plant implements AI that monitors sensor readings to alert engineers 48 hours before equipment failure. Which business value mechanism does this represent, and which industry use case is it?
APersonalization — the system tailors maintenance schedules to each machine's individual behavior profile.
BOptimization — Predictive Maintenance. The system improves operational decisions (when to service) beyond what manual inspection can achieve at scale.
CAutomation — Predictive Maintenance. The system fully replaces human engineers with AI decisions.
DOptimization — Process Optimization. The system improves manufacturing throughput by eliminating bottlenecks.

A healthcare provider wants to reduce the time physicians spend writing visit notes. They record provider-patient conversations and want structured clinical notes generated automatically. Which AWS service directly addresses this?
AAmazon Transcribe — converts audio to text so physicians can copy and paste the transcript into their notes system.
BAmazon Comprehend — extracts clinical entities from the transcript and formats them as structured data.
CAWS HealthScribe — purpose-built service that generates structured clinical notes from provider-patient conversations automatically.
DAmazon Textract — extracts text from scanned clinical forms and handwritten notes into machine-readable format.
MC 4MCC1
A self-driving car system must identify every pedestrian, vehicle, and road sign in its camera feed and determine precisely which pixels of the image belong to each object. Which Computer Vision task does this require?
AImage Classification — the system assigns a single label to the entire camera frame each second.
BObject Detection — the system draws bounding boxes around objects and labels each one.
CSegmentation — the system classifies every individual pixel to its corresponding object, providing the most granular spatial understanding.
DDimensionality Reduction — the system compresses the camera feed to essential features for faster processing.
MC 5MCC2–C3
An insurance company processes 50,000 scanned claim forms per day, each containing handwritten amounts and checkboxes. They want to extract the claim value and approval status from each form automatically. Which AWS service is most appropriate, and why?
AAmazon Comprehend — it performs entity recognition to extract monetary values and status labels from the raw text of claim forms.
BAmazon Textract — it extracts text, tables, and form fields from scanned documents using OCR, including handwritten fields.
CAmazon Rekognition — it analyzes document images to identify text regions and structure in scanned forms.
DAmazon Lex — it reads the scanned forms via its NLU engine and routes claims to the correct processing workflow.
MC 6MCC4

AThe ML system is always faster than rule-based filtering because neural networks process transactions in parallel.
BThe ML system learns what normal transaction behavior looks like and flags statistical deviations, catching novel fraud patterns that no predefined rule anticipated.
CThe ML system eliminates false positives entirely because it uses a larger dataset than any manually authored ruleset.
DThe ML system requires less maintenance because rules never need updating once the initial model is trained.
MC 7MCD1
A software team proposes using ML to determine whether a given number is even or odd. A senior engineer objects. Which response best explains why ML is inappropriate here?
AEven/odd detection requires a dataset too small to train an ML model with meaningful accuracy.
BThe rule is completely deterministic — divide by 2, check remainder. ML adds cost and complexity where a one-line formula is fully sufficient and always correct.
CML cannot process numerical inputs, making it structurally incompatible with integer parity detection.
DML would require reinforcement learning for a binary classification task, which is prohibitively expensive at this scale.
MC 8MCE1
A telecom company wants to predict how much data (in GB) a specific customer will consume next month based on their usage history and device type. Which ML technique and subtype applies?
AUnsupervised Clustering — group customers by usage pattern to estimate consumption for each segment.
BSupervised Classification — classify each customer into a usage tier (low / medium / high) and assign the tier's average consumption.
CSupervised Regression — predict a continuous numeric output (GB consumed) from labeled historical usage data.
DReinforcement Learning — the agent learns optimal consumption prediction through interaction with the customer's billing environment.
MC 9MCE1–E2

ASupervised Classification — train a model on labeled customer segments from a prior marketing study.
BUnsupervised Clustering — let the model discover natural groupings in unlabeled data without predefined categories.
CSupervised Regression — predict the revenue each customer will generate based on transaction history.
DDimensionality Reduction — compress the transaction features to find the most predictive variables for marketing.
MC 10MCE2
A data science team has a dataset with 800 features per sample. Visualizing and modeling it directly is computationally infeasible. They want to reduce it to its most meaningful components while preserving the underlying structure. Which unsupervised technique applies?
AClustering — group the 800 features into natural categories to reduce redundancy.
BDimensionality Reduction — reduce the feature space to its most meaningful components while preserving structural relationships.
CSupervised Regression — predict which features contribute most to the target variable and discard the rest.
DReinforcement Learning — the agent learns to select the most informative features through iterative reward signals.
MC 11MCE3

ALabeled demonstrations from human operators — the agent learned by imitating expert movements provided as training data.
BReward and penalty signals from environmental interaction — the agent developed a policy through trial and error with no labeled dataset.
CUnlabeled sensor readings — the agent clustered successful and unsuccessful movements to extract optimal motion patterns.
DA regression model predicting success probability per movement — the agent selected the highest-probability action each step.
MC 12MCF1
A content team of 5 writers produces 50 product descriptions per day. After deploying a Generative AI system, they produce 5,000 per day at the same quality level. Which Gen AI capability primarily explains this outcome?
AAdaptability — the model flexes across product categories without needing to be retrained for each one.
BCreativity — the model generates novel product descriptions that outperform what human writers produce.
CScalability — the model produces content volume at a speed no human team can match, multiplying output by 100×.
DData Efficiency — the model required only a small sample of existing descriptions to achieve high-volume production capability.
MC 13MCG1
A lawyer asks an AI assistant to cite a specific court ruling from 2019. The AI responds with a detailed, confident citation — including the case name, judge, and exact ruling language. The lawyer later discovers the case does not exist. What Generative AI risk does this exemplify?
ANondeterminism — the model generates inconsistent outputs when asked the same question multiple times.
BInterpretability — the model cannot explain which sources it used to construct the citation.
CHallucination — the model generated a plausible but entirely fabricated case with confident presentation.
DRegulatory Violation — the model exposed confidential legal data from its training corpus in the response.
MC 14MCG1–G2
A financial services firm deploys a Gen AI chatbot and discovers it occasionally produces different answers to the same client question across sessions. Which risk does this represent, and which mitigation strategy most directly addresses it?
AHallucination — addressed by implementing RAG to ground responses in factual retrieved documents.
BNondeterminism — addressed by monitoring outputs for consistency and setting lower temperature parameters to reduce response variation.
CToxicity — addressed by guardrails filtering harmful content categories from the response pipeline.
DInterpretability — addressed by enabling model logging to trace which training examples produced each output.
MC 15MCG2
A company trains a large language model on internal HR records containing employee salaries, performance reviews, and disciplinary history. Before deployment, which mitigation strategy directly reduces the risk that the model will reproduce this sensitive data in responses?
AModel Testing — evaluate the model on a sample of prompts before deployment to catch any undesired behavior patterns.
BGuardrails — filter outputs containing patterns matching employee names, salary figures, or performance data.
CData Anonymization — remove or mask personally identifiable information from the training data before the model is trained on it.
DMonitoring — continuously scan production outputs for sensitive data patterns and alert the security team when detected.

AStable Diffusion — its multimodal capability allows it to process both text queries and support documentation images.
BAmazon Titan — AWS-native model optimized for embeddings and semantic search across support documentation.

MC 17MCH1
A team evaluates two models for a real-time customer service chatbot: Model X achieves 96% accuracy but requires 800ms average response time. Model Y achieves 91% accuracy with 120ms response time. The customer SLA requires responses under 200ms. Which model should be selected and why?
AModel X — 96% accuracy is always preferable in customer-facing systems regardless of latency.
BModel Y — it meets the 200ms SLA constraint (120ms) while delivering acceptable 91% accuracy. Model X violates the deployment constraint regardless of accuracy.
CModel X — the latency gap between 800ms and 200ms can be addressed through infrastructure optimization without changing the model.
DNeither — a 5% accuracy gap between models is too large to accept and both should be rejected in favor of further development.

MC 19MCI1
An e-commerce platform deploys a Gen AI recommendation engine. The model achieves 94% precision on internal benchmarks. Three months post-deployment, revenue per user has not changed. What does this reveal about the success evaluation?
AThe model's precision metric is too low — 94% is insufficient for e-commerce recommendation systems at scale.
BTechnical performance is necessary but not sufficient. High model accuracy without business metric improvement (ARPU, Conversion Rate) indicates the model is not creating business value.
CRevenue is a lagging indicator — the platform should wait at least 12 months before concluding the model has no business impact.
DThe model requires fine-tuning — internal benchmark precision does not translate to production accuracy without domain-specific training data.
